 ## Інтелектуальна система управління особистими фінансами з прогнозуванням витрат

---

## 🔹 Опис проєкту

**Інтелектуальна система управління особистими фінансами з прогнозуванням витрат** — це програмний застосунок, розроблений мовою **Python**, який дозволяє користувачу:

- вести облік **доходів та витрат**
- переглядати баланс бюджету
- аналізувати витрати по категоріях
- переглядати графіки та статистику
- виконувати **прогнозування майбутніх витрат**
- експортувати дані у CSV файл

Проєкт реалізований у вигляді настільної програми з графічним інтерфейсом (**GUI**) на базі бібліотеки **CustomTkinter**.

---

## 🔹 Основні можливості програми

### ✅ 1. Облік доходів (Income)
Доходи — це всі надходження грошей користувача, наприклад:
- зарплата
- стипендія
- подарунки
- продаж речей
- будь-які інші фінансові надходження

Користувач може додавати дохід, вказуючи:
- категорію (наприклад: "Зарплата")
- суму
- дату (формат: `YYYY-MM-DD`)

---

### ✅ 2. Облік витрат (Expense)
Витрати — це всі фінансові витрати користувача, наприклад:
- їжа
- транспорт
- оренда
- покупки
- розваги

Користувач може додавати витрату, вказуючи:
- категорію (наприклад: "Їжа")
- суму
- дату (формат: `YYYY-MM-DD`)

---

### ✅ 3. Автоматичний баланс
Баланс розраховується автоматично за формулою:

**Баланс = Доходи – Витрати**

Якщо баланс негативний — це означає, що витрати перевищують доходи.

---

### ✅ 4. Аналітика витрат і доходів
Система автоматично:
- підраховує загальні доходи та витрати
- визначає найбільші категорії витрат
- будує статистику витрат по місяцях
- формує текстові рекомендації

---

### ✅ 5. Побудова графіків
Програма будує графіки:
- діаграма витрат по категоріях (Pie Chart)
- діаграма доходів по категоріях
- динаміка витрат по місяцях (Line Chart)
- ТОП категорій витрат (Bar Chart)

---

### ✅ 6. Прогнозування витрат
Оскільки тема диплому включає прогнозування витрат, у програмі реалізовано модуль прогнозування на основі історичних фінансових даних.

Прогноз дозволяє оцінити майбутні витрати користувача на основі попередніх записів.

(У майбутньому система може бути розширена та доповнена більш складними моделями прогнозування.)

---

### ✅ 7. Експорт даних у CSV
Користувач може експортувати всі транзакції у формат CSV, щоб:
- відкрити у Excel / Google Sheets
- виконати аналіз вручну
- зберігати резервну копію фінансових даних

```

├── requirements.txt # Список бібліотек для встановлення

└── .gitignore # Ігнорування службових файлів Git
```

## 🔹 Формат файлу `data.json`

Файл `data.json` використовується як локальна база даних.
Він містить всі транзакції у вигляді JSON структури.

Приклад структури:

```json
{
  "income": [],
  "expenses": []
}

Кожен запис доходу має вигляд:

{
  "amount": 5000,
  "category": "Зарплата",
  "date": "2026-05-01"
}

Кожен запис витрати має вигляд:

{
  "amount": 300,
  "category": "Їжа",
  "date": "2026-05-02"
}
```

🔹 Як встановити і запустити програму
✅ 1. Встановлення Python

Для запуску програми потрібно встановити Python версії 3.10+ (рекомендовано 3.12).

Перевірити версію Python можна командою:
```bash 
python --version
```

✅ 2. Встановлення залежностей

У папці з проєктом потрібно виконати команду:
```bash
python -m pip install -r requirements.txt
```
Ця команда автоматично встановить всі необхідні бібліотеки.

✅ 3. Запуск програми

Для запуску програми необхідно виконати:

python main.py

Після цього відкриється графічне вікно програми.

CSV файл можна використовувати для аналізу у Excel або Google Sheets.

---
🔹 Основні правила використання
✅ Правильне введення дати

✅ Правильне введення суми
Сума повинна бути: числом,  більше 0
Приклад:

1000.50
