# Інтелектуальна система управління фінансами

Програма дозволяє вести облік доходів і витрат та прогнозувати витрати на майбутні дні.

## Встановлення

1. Встановіть Python 3.8+.
2. Скачайте репозиторій або скопіюйте файли у будь-яку папку.
3. Переконайтеся, що всі файли знаходяться в одній директорії.

## Запуск

В терміналі виконайте команду:

```bash
python main.py
